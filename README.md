# Ceylon Peacock Travel Website

This is a premium travel agency website built with HTML, Tailwind CSS, and JavaScript.

## how to Run
Simply open `index.html` in any modern web browser.

## structure
- **index.html**: Home page with Hero section, popular tours, and features.
- **about.html**: Company information and mission.
- **tours.html**: Overview of all tour packages.
- **package1.html**: Details for "Heritage & Culture" tour.
- **package2.html**: Details for "Rest & Relax" tour.
- **package3.html**: Details for "Wild Safari" tour.
- **gallery.html**: Photo gallery of Sri Lanka.
- **services.html**: List of services provided.
- **contact.html**: Contact form and details.
- **testimonials.html**: Customer reviews.
- **images/**: Contains the SVG logo.

## technology
- **HTML5**: Semantic structure.
- **Tailwind CSS (CDN)**: Styling and responsive design.
- **JavaScript**: simple mobile menu interaction.

## note
The images are fetched from Unsplash. You will need an internet connection to see them initially.
